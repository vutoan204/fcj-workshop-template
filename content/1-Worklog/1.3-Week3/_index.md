### Week 3 Objectives:
* Deepen structural understanding of VPC design patterns, including routing logic with Route Tables, NAT Gateways, and Internet Gateways.
* Master granular infrastructure security mechanisms: effectively separating stateful firewall rules via Security Groups and stateless perimeter traffic filtering using Network ACLs.
* Build, map out, and configure a complete multi-tier custom VPC containing isolated Public subnets and Private subnets directly via the AWS Management Console.
