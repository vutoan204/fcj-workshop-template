# Internship Worklog Overview

This page records the chronological sequence of my 12-week internship journey at First Cloud AI Journey, transitioning from fundamental AWS cloud infrastructure configurations to developing the frontend user experience and integrating production cloud services for the **NodeJ2Car** auto parts platform.

The detailed weekly roadmap is outlined below:

**Week 1:** [Program onboarding, AWS Cloud fundamentals, and basic account security configuration](1.1-Week1/)

**Week 2:** [Advanced cloud security and permissions management (IAM Policies, MFA, and Security Groups)](1.2-Week2/)

**Week 3:** [Multi-Tier VPC networking setup (Subnets, NAT Gateways, and Route Tables)](1.3-Week3/)

**Week 4:** [Web application hosting and high-concurrency traffic routing (EC2 & Application Load Balancer)](1.4-Week4/)

**Week 5:** [Asynchronous messaging pipeline and MongoDB-compatible data tier (SQS & DocumentDB)](1.5-Week5/)

**Week 6:** [NodeJ2Car React SPA workspace initialization and development standards setup](1.6-Week6/)

**Week 7:** [UI layout designing and atomic component prototyping (Tailwind CSS & Radix UI)](1.7-Week7/)

**Week 8:** [Interactive 2D vehicle schematics browser and client-side state synchronization](1.8-Week8/)

**Week 9:** [E-Commerce payment gateways integration (VNPay, MoMo, Stripe) and API connections](1.9-Week9/)

**Week 10:** [Real-time client support widget connection using WebSockets / Socket.io](1.10-Week10/)

**Week 11:** [AI-powered diagnostic scanning engine integration via Bedrock and backend APIs](1.11-Week11/)

**Week 12:** [Edge caching acceleration (CloudFront CDN) and edge traffic filtering (AWS WAF)](1.12-Week12/)
