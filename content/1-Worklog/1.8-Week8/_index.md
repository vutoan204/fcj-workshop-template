### Week 8 Objectives:
* Implement secure application parameter and configuration storage using AWS Systems Manager Parameter Store and AWS Secrets Manager with automatic credential rotation.
* Overview international compliance standards (ISO 27001, PCI DSS) and explore threat detection mechanisms using Amazon GuardDuty.
* Compare AWS KMS data-at-rest encryption with AWS CloudHSM hardware security modules, and manage SSL/TLS certificates via AWS Certificate Manager (ACM).
* Analyze SSO solutions and user authentication patterns using Amazon Cognito, AWS IAM Identity Center, and AWS Directory Service.
* Conduct infrastructure vulnerability assessments using AWS Inspector, optimize cost and security with Trusted Advisor, and configure Lambda functions with encrypted database connection strings.
